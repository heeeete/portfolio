 일상을 기록하는 데 여러 앱을 사용해야 하는 불편함을 해소하기 위해 마모리를 개발했습니다. 해당 프로젝트는 frontEnd 두 명이서 진행을 했습니다. backEnd가 없어서 직접 데이터베이스를 만들어보고 AWS S3 스토리지를 사용해 보면서 많은 경험을 할 수 있었습니다.

 인생 첫 프로젝트를 시작하면서 Git으로 협업을 해보고, React-Native, Mysql, RestAPI 등 모든 것이 처음이라 힘든 부분이 정말 많았지만 CSS, JavaScript, React 등 웹 개발의 기본 지식을 습득할 수 있었고, 개발의 매력을 알게 된 저의 첫 번째 프로젝트입니다.

✅ 역할

  • Todo 알람 기능
  • 영화, 책 리뷰 기능
  • 로그인 유지 기능
  • 테마 기능

✅ 주요 기능

  • 다이어리
    • 날씨, 기분, 사진을 함께 첨부하여 작성. 최신 날짜 순으로 조회. 수정, 삭제 가능. 키워드 또는 날짜 검색을 통한 특정 다이어리 조회 가능.
  • 할 일 목록
    • 알람 설정 및 유지 옵션 선택 가능. 등록된 할 일 들은 순서 편집 및 수정, 삭제 가능
    • 유지 옵션 미선택 시 익일 오전 6시에 자동 삭제
  • 영화/책 리뷰 기능
    • 검색을 통해 영화/책 정보를 가져와 나의 리뷰와 함께 ‘별로예요’, ‘재밌어요’로 구분하여 등록. 수정, 삭제 가능. 키워드 검색을 통한 특정 리뷰 조회 가능.
  • 마이페이지
    • 이번 달 다이어리 감정 비율에 따라 감정 이미지를 나열 및 감정 멘트를 출력. 테마 색상 선택 가능. 나의 정보를 조회 및 수정 가능.
  • 한 달간의 감정 비율 확인
    • 한 달간의 감정 비율을 한눈으로 확인 가능
