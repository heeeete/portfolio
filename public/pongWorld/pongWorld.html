PongWorld

PongWorld 프로젝트는 외부 라이브러리 없이 Vanilla.js를 이용해 SPA 방식으로 개발한 실시간 PingPong 게임 플랫폼입니다.

 게임의 성능 최적화를 위해 CSS의 position 속성과 translate 속성을 비교 분석하였고, 이를 통해 쾌적한 게임을 만들 수 있었습니다.

 서버와 양방향 통신을 하기위해 처음으로 WebSocket을 사용해 보았습니다. 비동기적 특성과 복잡한 이벤트 처리에 어려움을 겪었지만, 성공적으로 실시간 웹 사이트를 만들 수 있었습니다. WebSocket을 사용해 본 것만으로도 의미가 있었지만 성공적으로 프로젝트를 마무리했다는 점에서 더욱 의미가 있습니다.

✅ 역할

  • SPA 구현
    • HTML history API를 사용하여 동적으로 컨텐츠 로드하고 URL 업데이트
    • 정규식을 이용해 URL을 매칭시키고 해당 URL에 맞는 DOM을 동적으로 렌더링
  • 실시간 게임 기능
    • 탁구채 조작을 마우스, 키보드 두가지로 할 수 있게 구현
    • 부드러운 애니메이션을 위해 requestAnimationFrame()을 사용
  • 실시간 채팅 기능
    • WebSocket을 사용하여 실시간 채팅 기능 구현
    • 서버와 협업으로 무한 스크롤 기능 구현
  • 로그인 기능
    • accessToken은 XSS 공격으로부터 보호하기 위해 메모리에 저장
    • refreshToken은 CSRF 공격 및 토큰 탈취 방지를 위해 Secure, HttpOnly, SameSite 옵션이 설정된 쿠키에 저장
  • 마이페이지
    • REST API를 사용하여 사용자 정보 업데이트

✅ 주요 기능

  • 실시간 핑퐁 게임
  • 토너먼트 게임
  • 실시간 채팅
  • 친구 관리(차단, 친구 추가, 삭제)
  • 마이페이지

⭐ 성과

  • 애니메이션 최적화 : 프레임 드랍 발생 빈도가 크게 감소하여, 게임의 FPS가 평균 59fps로 유지되며, 최저 FPS가 58fps로 이전 15fps에서 크게 개선
